# 🎮 GameNest

Aplicación web para gestión y comunidad de videojuegos desarrollada como parte del Sprint 1.

---

## 📌 Descripción

GameNest es una aplicación web que permite a los usuarios registrarse e iniciar sesión para acceder a una plataforma gamer donde pueden:

- Descubrir nuevos videojuegos  
- Conectar con otros usuarios  
- Participar en experiencias interactivas  

El sistema cuenta con validaciones en tiempo real, autenticación segura y una interfaz moderna.

---

## ⚙️ Tecnologías utilizadas

Frontend:
- React (Vite)
- CSS moderno
- React Icons

Backend:
- Node.js
- Express
- bcrypt

Base de datos:
- SQLite

---

## 🚀 Ejecución del proyecto

1. Clonar repositorio

git clone https://github.com/JorgeBosquez/gamenest.git
cd gamenest

---

2. Backend

cd backend
npm install
npm run dev

Servidor:
http://localhost:3000

---

3. Frontend

cd ../frontend
npm install
npm run dev

Aplicación:
http://localhost:5173
